## Github Finder

App to search Github users. Helps to search for users globally.

<img src="images/github-finder.gif">

#

## Getting Started

Clone the repo:

```bash
git clone
cd rdc-x
```

Install dependencies and run server:

```bash
npm install
npm start
```

Run Unit Test

```bash
npm run test
```

Run Unit Test Coverage

```bash
npm run test -- --coverage --watchAll=false
```

Build

```bash
npm run build
```
